# Vityarthi Library Management System

## Overview
Vityarthi Library Management System is a Java-based console application developed to simplify and manage day-to-day library operations. The system allows users to add books, manage members, issue and return books, and store records efficiently.

This project demonstrates the use of Object-Oriented Programming concepts, Java Collections Framework, Exception Handling, and File Handling using Serialization.

---

## Features
- Add new books and members
- Search books and members
- Issue books to members
- Return books with proper validation
- Prevent duplicate entries and invalid operations
- Save and load records using Java Serialization
- User-friendly console-based interface

---

## Technologies Used
- Java (JDK 8+)
- Object-Oriented Programming (OOP)
- Java Collections Framework

- File Handling and Serialization
